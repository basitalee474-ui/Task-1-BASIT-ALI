# Number Guessing Game

## Project Overview
A modern Java Swing based Number Guessing Game featuring multiple difficulty levels, score tracking, achievements, statistics management, and an attractive user interface.

## Project Structure

src/numberguess/
├── Main.java                 # Application entry point
├── NumberGameGUI.java        # Main graphical user interface
├── GameLogic.java            # Core game logic and validations
├── Player.java               # Player data management
├── AchievementManager.java   # Achievement and badge system
├── ScoreManager.java         # High score management
├── StatisticsManager.java    # Game statistics tracking
└── ThemeManager.java         # UI colors and themes

## Features

- Interactive Java Swing GUI
- Random Number Generation
- Multiple Difficulty Levels
- High Score Leaderboard
- Achievement System
- Statistics Tracking
- Modern Theme Design
- File-based Data Storage

## Requirements

- Java JDK 8 or higher
- IntelliJ IDEA (Recommended)

## How to Run

1. Open the project in IntelliJ IDEA.
2. Navigate to `Main.java`.
3. Click Run.
4. Start guessing numbers and enjoy the game.

## Data Storage

Game data is stored locally using text files:
- scores.txt
- statistics.txt
